wzbrowser

Browse MapleStory WZ files entirely in your browser.

Launch wzbrowser

What It Does

MapleStory stores much of its game data in WZ archive files. Traditionally you needed desktop tools to open them. wzbrowser brings that workflow to the browser: load a WZ file from your computer, explore its contents in a tree view, preview images on a canvas, and export what you need.

Key Features

  • Local-only processing - files never leave your machine
  • WZ decoding in the browser - no install or server upload
  • XML export - pull structured data from nodes
  • PNG export - save sprites and images directly
  • Canvas preview - inspect graphics before exporting

Who It's For

Modders, dataminers, artists referencing sprites, and curious players who want to understand how MapleStory packages its assets. Because everything runs client-side, it's also a safer option when you don't want to upload proprietary game files anywhere.

How to Use It

  1. Open wzbrowser - click Launch wzbrowser above. It runs entirely in your browser.
  2. Load a WZ file - choose a .wz file from your local MapleStory installation. The file is read in place and never uploaded.
  3. Explore the tree - browse the archive's node structure in the tree view to find sprites, sounds, and property data.
  4. Preview assets - click an image node to render it on the canvas before you decide to export.
  5. Export what you need - save individual images as PNG or pull structured node data as XML.

Since decoding happens client-side, large archives stay on your machine and nothing about your files is sent anywhere.

FAQ

Where do I get WZ files?

WZ files come from your local MapleStory installation. You are responsible for complying with the game's terms of service when extracting or using game assets.

Are my files uploaded?

No. wzbrowser decodes files entirely in your browser using JavaScript. Nothing is sent to a server.

What formats are supported?

The tool focuses on standard MapleStory WZ structures. Some encrypted or version-specific files may require updates as the game patches.
